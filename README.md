Nesse projeto, criei uma landing page para uma rede de hotelaria com uma proposta bem interessante: integrar hotéis à natureza, proporcionando uma experiência que une conforto e sustentabilidade 🌿🏨

A principal intenção foi aplicar, na prática, tudo o que foi aprendido ao longo do curso — desde a construção de layouts até a criação de uma interface moderna, responsiva e bem estruturada.

Um dos pontos de destaque foi a abordagem mobile first, garantindo que a experiência fosse otimizada desde dispositivos móveis até telas maiores, algo essencial no desenvolvimento moderno.

Esse processo foi fundamental para fixar os conhecimentos adquiridos e aumentar minha confiança no uso do Tailwind, principalmente na criação de componentes e na otimização do fluxo de desenvolvimento.
